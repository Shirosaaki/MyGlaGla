## 🎯 Abdeckung

### Dokumentierte Sprachmerkmale

**Datentypen:**
- ✅ Ganzzahlen (`int`)
- ✅ Fließkommazahlen (`float`)
- ✅ Zeichenfolgen (`string`)
- ✅ Zeichen (`char`)
- ✅ Boolesche Werte (`bool`)
- ✅ Leer (`void`)
- ✅ Arrays (`type[]`)
- ✅ Zeiger (`type*`)
- ✅ Strukturen (`destruct`)
- ✅ Enums (`desnum`)

**Schlüsselwörter & Funktionen:**
- ✅ `Deschodt` - Funktionsdefinition
- ✅ `Eric` - Hauptfunktion / Variablendeklaration
- ✅ `peric` - Ausgabe drucken
- ✅ `deschodt` - Rückgabewert
- ✅ `erif` - If-Bedingung
- ✅ `deschelse` - Else-Bedingung
- ✅ `aer` - For-in-Schleife
- ✅ `darius` - While-Schleife
- ✅ `deschontinue` - Continue-Anweisung
- ✅ `deschreak` - Break-Anweisung
- ✅ `&` - Adressoperator (Zeiger)
- ✅ `*` - Dereferenzierungsoperator (Zeiger)

**Operatoren:**
- ✅ Arithmetik: `+`, `-`, `*`, `/`, `mod`, `div`
- ✅ Vergleich: `==`, `!=`, `<`, `>`, `<=`, `>=`
- ✅ Logisch: `&&`, `||`, `!`
- ✅ Typannotationen: `->`

## 🔄 Interaktiver REPL

Alle Beispiele können im interaktiven REPL getestet werden:

```bash
stack run
# or
./glados
```

**REPL-Funktionen:**
- Sofortige Ausführung in einer Zeile: Tippen und Enter drücken
- Mehrzeilige Blöcke: Verwenden Sie `:code` ... `:end` Trennzeichen
- Zeilenbearbeitung: Vollständige Unterstützung für die Bearbeitungshistorie
- Geben Sie einen beliebigen Ausdruck zur Auswertung ein

## 📝 Beispiel-Sitzung

```
$ stack run
glados> eric x = 10 -> int
10
glados> peric("x = {x}")
x = 10
glados> :code
code> eric factorial(n -> int) -> int
code>     erif (n <= 1):
code>         deschodt 1
code>     deschelse:
code>         deschodt n * factorial(n - 1)
code> :end
<closure>
glados> peric("5! = {factorial(5)}")
5! = 120
```
