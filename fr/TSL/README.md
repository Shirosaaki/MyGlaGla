## 🎯 Couverture

### Fonctionnalités du langage documentées

**Types de données:**
- ✅ Integers (`int`)
- ✅ Floats (`float`)
- ✅ Strings (`string`)
- ✅ Characters (`char`)
- ✅ Booleans (`bool`)
- ✅ Void (`void`)
- ✅ Tableaux (`type[]`)
- ✅ Pointeurs (`type*`)
- ✅ Structures (`destruct`)
- ✅ Énumérations (`desnum`)

**Keywords & Fonctions:**
- ✅ `Deschodt` - Définition de fonction
- ✅ `Eric` - Fonction principale / Déclaration de variable
- ✅ `peric` - Imprimer la sortie
- ✅ `deschodt` - Instruction de retour
- ✅ `erif` - Conditionnel If
- ✅ `deschelse` - Conditionnel Else
- ✅ `aer` - Boucle For-in
- ✅ `darius` - Boucle While
- ✅ `deschontinue` - Instruction Continue
- ✅ `deschreak` - Instruction Break
- ✅ `&` - Opérateur d'adresse (pointeurs)
- ✅ `*` - Opérateur de déréférencement (pointeurs)

**Operators:**
- ✅ Arithmetic: `+`, `-`, `*`, `/`, `mod`, `div`
- ✅ Comparison: `==`, `!=`, `<`, `>`, `<=`, `>=`
- ✅ Logical: `&&`, `||`, `!`
- ✅ Type annotations: `->`

## 🔄 REPL interactif

Tous les exemples peuvent être testés dans le REPL interactif :

```bash
stack run
# or
./glados
```

**Fonctionnalités REPL :**
- Exécution immédiate sur une seule ligne : tapez et appuyez sur Entrée
- Blocs multilignes : utilisez les délimiteurs `:code` ... `:end`
- Édition de ligne : prise en charge complète de l'historique d'édition
- Tapez n'importe quelle expression à évaluer

## 📝 Exemple de session

```
$ stack run
glados> eric x = 10 -> int
10
glados> peric("x = {x}")
x = 10
glados> :code
code> eric factorial(n -> int) -> int
code>     erif (n <= 1):
code>         deschodt 1
code>     deschelse:
code>         deschodt n * factorial(n - 1)
code> :end
<closure>
glados> peric("5! = {factorial(5)}")
5! = 120
```
